# GamemodeSwitch
Switch your gamemode in Minecraft easier

# Commands:
- /gamemode 0
- /gamemode 1
- /gamemode 2
- /gamemode 3
- /gamemode survival
- /gamemode creative
- /gamemode adventure 
- /gamemode spectator
- /gamemode reload
- /gm 0
- /gm 1
- /gm 2
- /gm 3
- /gm survival
- /gm creative
- /gm adventure
- /gm spectator
- /gm reload
- /s
- /c
- /a
- /sp

# Permissions:
- gamemodeswitch.change
- gamemodeswitch.admin.reload
